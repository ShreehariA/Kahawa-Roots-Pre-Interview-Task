\small
\begin{columns}[T]

\begin{column}{0.55\textwidth}

\textbf{\faRoute\ The Journey in 4 Steps}
\vspace{0.2em}

\begin{enumerate}\setlength{\itemsep}{2pt}
    \item \textbf{Data comes in} — Farmer or admin enters details via web form or CSV
    \item \textbf{Validated \& cleaned} — Every field checked automatically; invalid rows rejected, logged for review
    \item \textbf{Stored safely} — Raw backup kept untouched, clean data in DB with timestamps \& GPS at each stage
    \item \textbf{Customers see it} — Each shipment gets a QR code linking to a public page with map, timeline \& farmer info
\end{enumerate}

\vspace{0.2em}
{\scriptsize\textit{Data flows in → validation → storage → accessible via QR.}}

\begin{frame}{Why These Choices Work for Kahawa Roots}
\scriptsize
\begin{columns}[T]

\begin{column}{0.48\textwidth}

\textbf{\faPoundSign\ Cheap at Small Scale}
\vspace{0.1em}

\begin{itemize}\setlength{\itemsep}{3pt}
    \item \textbf{\textcolor{kahawaBrown}{Cloud Run}} scales to zero — when nobody's using it, you pay nothing. No idle infrastructure costs.

    \item \textbf{\textcolor{kahawaBrown}{Cloud Functions}} only run when data arrives — a few pence per execution, not a monthly server bill.

    \item \textbf{\textcolor{kahawaBrown}{GCP}} — eligible for startup credits (up to \$200K). The platform could run for months before costing anything.

    \item \textbf{\textcolor{kahawaBrown}{PostgreSQL}} — free-tier eligible, battle-tested, no licensing fees.
\end{itemize}

\vspace{0.1em}
\textbf{\faTools\ Easy to Maintain}
\vspace{0.1em}

\begin{itemize}\setlength{\itemsep}{3pt}
    \item \textbf{\textcolor{kahawaBrown}{3 small apps}} instead of 1 big one — if the QR page needs updating, the admin panel stays untouched.

    \item \textbf{\textcolor{kahawaBrown}{Infra as code}} — the entire system is defined in config files. If something breaks, one command rebuilds it.

    \item \textbf{\textcolor{kahawaBrown}{Auto-deploy}} — push code → it's tested and live. No manual uploads, no human error.
\end{itemize}

\end{column}

\begin{column}{0.48\textwidth}

\textbf{\faShieldAlt\ Safe \& Reliable}
\vspace{0.1em}

\begin{itemize}\setlength{\itemsep}{3pt}
    \item \textbf{\textcolor{kahawaBrown}{Raw data is never deleted}} — every CSV and form submission is backed up untouched. If anything goes wrong, the original data is always there.

    \item \textbf{\textcolor{kahawaBrown}{Two-step validation}} — data is checked at entry \textit{and} again before it reaches the database. Bad rows are rejected with clear error messages.

    \item \textbf{\textcolor{kahawaBrown}{Passwords stored securely}} — database credentials and API keys live in a vault, never in the code.

    \item \textbf{\textcolor{kahawaBrown}{UK data centre}} — everything runs in London (europe-west2). Data stays in the UK, GDPR-friendly.
\end{itemize}

\vspace{0.1em}
\textbf{\faChartLine\ Scales When You're Ready}
\vspace{0.1em}

\begin{itemize}\setlength{\itemsep}{3pt}
    \item \textbf{\textcolor{kahawaBrown}{BigQuery}} ready — when there's enough data, plug in analytics without rebuilding anything.

    \item \textbf{\textcolor{kahawaBrown}{Dashboard}} — Looker Studio for founders to see trends without touching the database.
\end{itemize}

\end{column}

\end{columns}

\end{frame}
